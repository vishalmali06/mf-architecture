# Micro-Frontend Architecture using Qiankun

This repository contains a micro-frontend architecture built using React and Qiankun.

## Applications
- main-app : Qiankun container application
- profile-app : Public profile application
- dashboard-app : Authenticated dashboard application

## Tech Stack
- React
- Qiankun
- Node.js
- MongoDB (for dashboard-app backend)

## Run Locally

### Main App
cd main-app
npm install
npm start

### Profile App
cd profile-app
npm install
npm run dev

### Dashboard App
cd dashboard-app
npm install
npm start
